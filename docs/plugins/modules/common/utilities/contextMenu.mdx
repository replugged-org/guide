### Functions

| Name       | Parameters                                                                                                                                                                                                                                                                    | Return Type | Description                      |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------- | -------------------------------- |
| `close`    |                                                                                                                                                                                                                                                                               | void        | Closes the current context menu  |
| `open`     | <Required /> `event`: MouseEvent <br /> `render`: (props: [`ContextMenuProps`](#ContextMenuProps)) => ReactNode <br /> `options`: [`ContextMenusOptions`](#ContextMenusOptions) <br /> `renderLazy`: Promise\<(props: [`ContextMenuProps`](#ContextMenuProps)) => ReactNode)> | void        | Opens a context menu             |
| `openLazy` | <Required /> `event`: MouseEvent <br /> `renderLazy`: Promise\<(props: [`ContextMenuProps`](#ContextMenuProps)) => ReactNode)> <br /> `options`: [`ContextMenusOptions`](#ContextMenusOptions)                                                                                | void        | Opens a lazy loaded context menu |

### Types \{#contextMenu-types}

#### `ContextMenuProps` \{#ContextMenuProps}

| Name             | Type                                                  | Default | Description                         |
| ---------------- | ----------------------------------------------------- | ------- | ----------------------------------- |
| `className`      | string                                                |         | Component class name                |
| `config`         | [`ContextMenusOptions`](#ContextMenusOptions)         |         | Options used for that context menu  |
| `context`        | string                                                | `APP`   | App context                         |
| `onHeightUpdate` | () => void                                            |         | Function ran to update the position |
| `position`       | [`Tooltip.Positions`](../../components/Tooltip#enums) |         | Context menu position               |
| `target`         | HTMLElement                                           |         | Context menu children element       |
| `theme`          | string                                                |         | Current theme                       |

#### `ContextMenusOptions` \{#ContextMenusOptions}

| Name                   | Type                                                  | Default | Description                                                            |
| ---------------------- | ----------------------------------------------------- | ------- | ---------------------------------------------------------------------- |
| `align`                | [`Tooltip.Aligns`](../../components/Tooltip#enums)    |         | Context menu alignment to the children; works together with `position` |
| `enableSpellCheck`     | boolean                                               | `false` | Whether to enable spell checking                                       |
| `impressionName`       | string                                                |         | Impression name for analytics                                          |
| `impressionProperties` | `ImpressionProperties`                                |         | Impression properties for analytics                                    |
| `onClose`              | () => void                                            |         | Function ran when the context menu got closed                          |
| `position`             | [`Tooltip.Positions`](../../components/Tooltip#enums) |         | Context menu position; works together with `align`                     |
